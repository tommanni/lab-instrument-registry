# Instructions

The instructions for running the backend can be found in the folder instructions/  
In the folder there are files django-virtualenv.md and database.md in Finnish and English with corresponding prefixes (fi- and en-)

## django-virtualenv.md

This file contains instructions to create and activate the virtual environment.

On top of that there are instructions for running the local Django server.

## database.md

In this file there are instructions to create a PostgreSQL database and configure Django to use that database.

Also contains database CLI instruction descriptions.
